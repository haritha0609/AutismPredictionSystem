# 🧩 Early Autism Prediction System

An AI-powered web application for **early autism screening** using both **behavioral assessment** and **facial image analysis**. The system combines Machine Learning and Deep Learning models to provide preliminary screening results along with wellness guidance.

> **Note:** This application is intended for educational and preliminary screening purposes only. It is **not** a substitute for professional medical diagnosis.

---

## 📌 Features

- 📝 Behavioral Assessment Questionnaire
- 🤖 Machine Learning-based Behavioral Prediction
- 📷 Facial Image Analysis using ResNet50
- 🧠 AI-powered Autism Screening
- 💡 Personalized Wellness Guidance
- 🎨 Modern Responsive User Interface
- 🔒 Face Detection Validation (Only facial images are accepted)

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AutismPredictionSystem.git
```

### 2️⃣ Move into the project folder

```bash
cd AutismPredictionSystem
```

### 3️⃣ Create a virtual environment

```bash
python -m venv venv
```

### 4️⃣ Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 5️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 🧠 Models Used

### Behavioral Prediction Model

- Algorithm: XGBoost Classifier
- Input: Behavioral questionnaire responses
- Output: Preliminary behavioral autism screening

### Facial Image Prediction Model

- Model: ResNet50 (Transfer Learning)
- Framework: TensorFlow/Keras
- Test Accuracy: **90.5%**
